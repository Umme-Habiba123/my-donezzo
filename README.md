# Donezo Dashboard - React

Project management dashboard built with React, Tailwind CSS, and DaisyUI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Note

If you encounter PowerShell execution policy errors on Windows, run:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Or use the HTML file directly by renaming `index-react.html` to `index.html`.
